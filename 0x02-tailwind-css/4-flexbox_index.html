Responsive Flexbox Layout

This layout uses Flexbox to show two columns on desktop, which stack vertically on mobile screens.

Content Section

This is the main content area. It takes up the remaining two-thirds of the screen width on larger devices.

When you resize the window to less than 768px, the `main` container's `flex-direction` changes to `column`, causing the sidebar and content to stack vertically, with each taking up the full width.
